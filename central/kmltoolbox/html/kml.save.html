kml.save

Save and run the current kml file, opening it in the default kml viewer.

Contents

  • Syntax
  • Description
  • Example

Syntax

k.save;
k.save('filename');

Description

kml.save saves the content of any plot created inside the kml object to an xml file representing these plots. If the name of the kml was not specified when creating it, it will save to a temporary file name. You can also specify the desired filename by passing the parameter filename when calling kml.save.

Example

% Create a new kml object
k = kml('my kml file');

% Create a sample plot inside the kml
t = linspace(0,360,1000);
k.plot(30*t, 90*cosd(t/2));

% Save the kml to the file sample.kml in the current path
k.save('sample.kml');

% Show the contents of the saved file in the MATLAB editor
edit('sample.kml');
